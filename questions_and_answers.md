
<p>

###### 4. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript
function sayHi(name) {
  return `Hi there, ${name}`;
}

console.log(sayHi());
```

- A: `Hi there,`
- B: `Hi there, undefined`
- C: `Hi there, null`
- D: `ReferenceError`

<details><summary><b>Answer</b></summary>
<p>

#### Answer: `Hi there, undefined`

<i>Here the sayHi function is being called without parameters (sayHi()), due to which the value of the name parameter is not set to any value, due to which the value of the name parameter is becoming undefined.</i>

</p>
</details>

###### 5. Write the `correct answer` from the following options and give an explanation (2-5 lines).

```javascript

const nums = [0, 1, 2, 3];

nums.forEach((num) => {
  if (num) count += 1;
});

console.log(count);
```

- A: 1
- B: 2
- C: 3
- D: 4

<details><summary><b>Answer</b></summary>
<p>

#### Answer: 3 ?

<i>There is no condition given 0 which causes elements with value equal to 0 to be dropped. If the condition is true for all other elements, the count variable is incremented by adding the total number of elements.</i>

</p>
</details>
